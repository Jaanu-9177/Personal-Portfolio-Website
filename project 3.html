Vulnerability Assessment On Windows

Project Overview

Vulnerability assessment in terms of cyber security can be understood as the process of Identifying, Enumerating and Ranking the vulnerabilities present in a system or network in order to patch them. It is concerned with the security of the resource and its environment and is a proactive approach.

Project Details

In this project, we discuss about a Vulnerability Assessment on Windows. In this A vulnerability assessment involves using scanning tools to gather information from devices on a network, such as software versions, and compare the information to known software vulnerabilities. Organizations typically run scans at set times and scheduled intervals for patching and upgrades.
